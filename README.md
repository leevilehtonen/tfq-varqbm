# tfq-varqbm
Variational quantum Boltzmann machine and variational quantum Imaginary time evolution built with Cirq and Tensorflow Quantum.

## Main references
* [Variational quantum Boltzmann machines](https://doi.org/10.1007/s42484-020-00033-7)
* [Variational ansatz-based quantum simulation of imaginary time evolution](https://doi.org/10.1038/s41534-019-0187-2)
* [Simultaneous Perturbation Stochastic Approximation of the Quantum Fisher Information](https://arxiv.org/abs/2103.09232)
* [TensorFlow Quantum: A Software Framework for Quantum Machine Learning](https://arxiv.org/abs/2003.02989)
* [Cirq](https://github.com/quantumlib/Cirq)
